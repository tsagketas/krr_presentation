# LaTeX Presentation (Beamer)

## Build

From this folder:

```powershell
xelatex -interaction=nonstopmode presentation.tex
xelatex -interaction=nonstopmode presentation.tex
```

Output: `presentation.pdf`

## Notes

- Designed for **XeLaTeX (MiKTeX)** and Greek+English text.
- Uses `tikz` for simple diagrams (no external images required).
